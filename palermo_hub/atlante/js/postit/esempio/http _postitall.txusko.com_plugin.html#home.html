                        <h2>Getting started</h2><hr />
                        <h4>Download</h4>
                        <p><a href="">Download the plugin</a> from the GitHub repository and include the css and the js files as usual.</p>
<pre class="prettyprint lang-html">
&lt;!-- CSS --&gt;
&lt;link rel="stylesheet" href="dist/jquery.postitall.css"&gt;

&lt;!-- JS --&gt;
&lt;script src="libs/jquery/dist/jquery.min.js"&gt;&lt;/script&gt;
&lt;script src="dist/jquery.postitall.js"&gt;&lt;/script&gt;
</pre>
                        <h4>Plugins</h4>
                        <p>Postitall can work alone but you can add some plugins to gain extra features and usability.</p>
                        <p>This is the list of supported plugins:</p>
                        <ul>
                            <li><a href="https://jqueryui.com/" target=_blank>Jquery UI</a> : The jQuery User Inteface. With this plugin you will improve notes drag and resize usage.</li>
                            <li><a href="http://alex-d.github.io/Trumbowyg/" target=_blank>Trumbowyg</a> : A lightweight WYSIWYG editor. With this plugin you will be able to edit notes content with an integrated html5 editor.</li>
                            <li><a href="http://labs.abeautifulsite.net/jquery-minicolors/" target=_blank>Minicolors</a> : A Tiny Color Picker. With this plugin you will be able to change background and text note color with a color picker.</li>
                        </ul>
<pre class="prettyprint lang-html">
&lt;!-- CSS --&gt;
&lt;link rel="stylesheet" href="libs/jquery-ui/themes/base/jquery-ui.css"&gt;
&lt;link rel="stylesheet" href="libs/trumbowyg/dist/ui/trumbowyg.css"&gt;
&lt;link rel="stylesheet" href="libs/minicolors/jquery.minicolors.css"&gt;
&lt;link rel="stylesheet" href="dist/jquery.postitall.css"&gt;

&lt;!-- JS --&gt;
&lt;script src="libs/jquery/dist/jquery.min.js"&gt;&lt;/script&gt;
&lt;script src="libs/jquery-ui/jquery-ui.min.js"&gt;&lt;/script&gt;
&lt;script src="libs/trumbowyg/dist/trumbowyg.min.js"&gt;&lt;/script&gt;
&lt;script src="libs/minicolors/jquery.minicolors.min.js"&gt;&lt;/script&gt;
&lt;script src="dist/jquery.postitall.js"&gt;&lt;/script&gt;
</pre>
<p class="text-muted">* Note that the css file <span class="label label-default">jquery.postitall.css</span> overrides some css elements of those plugins for a correct integration with postitall.</p>

                        <h4>Install with npm</h4>
                        <p>You can install PostItAll by using <a href="https://www.npmjs.com/">npm</a></p>
<pre class="prettyprint lang-bsh">
npm install postitall
</pre>
                        <p>Plugins:</p>
<pre class="prettyprint lang-sh">
npm install jquery-ui trumbowyg jquery-minicolors
</pre>
                        <p class="text-muted">* Remember to change references to your npm destination directory</p>

                        <h4>Install with bower</h4>
                        <p>Also, you can install PostItAll using <a href="http://bower.io/">Bower</a></p>
<pre class="prettyprint lang-bsh">
$ bower install PostItAll
</pre>
                        <p>Plugins:</p>
<pre class="prettyprint lang-sh">
$ bower install jquery-ui trumbowyg minicolors
</pre>
                        <p class="text-muted">* Remember to change references to your bower destination directory</p>

                        <a id="doc_globalconfig" class="anchor">&nbsp;</a>
                        <h2>Global configuration</h2>
                        <hr>
                        <div>
                            <p>The
 notes have some features that can be changed to satisfy your needs. By 
default all of them are enabled, but you can change any of these "flags"
 in the beginning of your scripts and all new notes will be affected by 
these changes.</p>
                            <p>This are the variables with default values:</p>
<pre class="prettyprint lang-js">
//Global vars : enable and disable features and change the notes behaviour
$.fn.postitall.globals = {
    prefix          : '#PIApostit_',//Id note prefixe
    filter          : 'domain',     //Options: domain, page, all
    savable         : false,        //Save postit in storage
    randomColor     : true,         //Random color in new postits
    toolbar         : true,         //Show or hide toolbar
    autoHideToolBar : true,         //Animation effect on hover over postit showing/hiding toolbar options
    removable       : true,         //Set removable feature on or off
    askOnDelete     : true,         //Confirmation before note removal
    draggable       : true,         //Set draggable feature on or off
    resizable       : true,         //Set resizable feature on or off
    editable        : true,         //Set contenteditable and enable changing note content
    changeoptions   : true,         //Set options feature on or off
    blocked         : true,         //Postit can not be modified
    minimized       : true,         //true = minimized, false = maximixed
    expand          : true,         //Expand note
    fixed           : true,         //Allow to fix the note in page
    addNew          : true,         //Create a new postit
    showInfo        : true,         //Show info icon
    pasteHtml       : true,         //Allow paste html in contenteditor
    htmlEditor      : true,         //Html editor (trumbowyg)
    autoPosition    : true,         //Automatic reposition of the notes when user resizes the screen
    addArrow        : 'back'        //Add an arrow to notes : none, front, back, all
};
</pre>
                        </div>

                        <h2>Note configuration</h2>
                        <hr>
                        <div>
                            <p>This are the properties and event callbacks for each new note:</p>
<pre class="prettyprint lang-js">
//Note global vars : Properties, style, features and events of the note
$.fn.postitall.defaults = {
    //Note properties
    id              : "",                       //Note id
    created         : Date.now(),               //Creation date
    domain          : window.location.origin,   //Domain in the url
    page            : window.location.pathname, //Page in the url
    osname          : navigator.appVersion,     //Browser informtion &amp; OS name
    content         : '',                       //Content of the note (text or html)
    position        : 'absolute',               //Position absolute or fixed
    posX            : '10px',                   //x coordinate (from left)
    posY            : '10px',                   //y coordinate (from top)
    right           : '',                       //x coordinate (from right). This property invalidate posX
    height          : 200,                      //Note total height
    width           : 160,                      //Note total width
    minHeight       : 210,                      //Note resizable min-width
    minWidth        : 170,                      //Note resizable min-height
    oldPosition     : {},                       //Position when minimized/collapsed (internal use)
    //Config note style
    style : {
        tresd           : true,                 //General style in 3d format
        backgroundcolor : '#FFFA3C',            //Background color in new postits when randomColor = false
        textcolor       : '#333333',            //Text color
        textshadow      : true,                 //Shadow in the text
        fontfamily      : 'verdana',            //Default font
        fontsize        : 'small',              //Default font size
        arrow           : 'none',               //Default arrow : none, top, right, bottom, left
    },
    //Enable / Disable features
    features : $.fn.postitall.globals,          //By default, copy of global defaults
    //Note flags
    flags : {
        blocked         : false,                //If true, the note cannot be edited
        minimized       : false,                //true = Collapsed note / false = maximixed
        expand          : false,                //true = Expanded note / false = normal
        fixed           : false,                //Set position fixed
        highlight       : false,                //Higlight note
    },
    //Attach the note to al html element
    attachedTo : {
        element         : '',                   //Where to attach
        position        : 'right',              //Position relative to elemente : top, right, bottom or left
        fixed           : true,                 //Fix note to element when resize screen
        arrow           : true,                 //Show an arrow in the inverse position
    },
    // Callbacks / Event Handlers
    onCreated: function(id, options, obj) { return undefined; },    //Triggered after note creation
    onChange: function (id) { return undefined; },                  //Triggered on each change
    onSelect: function (id) { return undefined; },                  //Triggered when note is clicked, dragged or resized
    onDblClick: function (id) { return undefined; },                //Triggered on double click
    onRelease: function (id) { return undefined; },                 //Triggered on the end of dragging and resizing of a note
    onDelete: function (id) { return undefined; }                   //Triggered when a note is deleted
};
</pre>

                        <h2>Functions / Methods</h2>
                        <hr>
                        <p>As said before, you have two ways to call note actions: by calling global or plugin methods.</p>
                        <p>This is the relation of all the actions with their calls and a brief description of each one:</p>
                        <div class="row">
                            <div class="col-md-12">
                                <table class="table table-striped">

                                        <tr>
                                            <th scope="row">New note</th>
                                            <td>
                                                $.PostItAll.new();<br>
                                                ...<br>
                                                $.PostItAll.new(content, options, object, callback);
                                            </td>
                                            <td>$(id).postitall();<br>
                                                ...<br>
                                                $(id).postitall('new', options, callback);</td>
                                            <td>Create a new note.
                                                <br><span class="label label-info">Global call</span> <b>content</b> is the note content and <b>object</b> is the element in the DOM.
                                                <br>For the two possibilities: <b>options</b> is an object with the new <a href="#doc_noteconfiguration">note options</a> and <b>callback</b> is the user function called when the new note is created.
                                                <br><a href="#doc_newnote">View an example</a>
                                            </td>
                                        </tr>

                                            <th scope="row">Change note options</th>
                                            <td>
                                                $.PostItAll.options(options);<br>
                                                $.PostItAll.options(id,options);
                                            </td>
                                            <td>
                                                $(id).postitall('options');<br>
                                                $(id).postitall('options',options);
                                            </td>
                                            <td>Get or set the options of the created notes.
                                                <br><span class="label label-info">Global call</span> In case you don't specify the note <b>id</b> the options changes affect all created notes.
                                                <br><b>options</b> is an object with the <a href="#doc_noteconfiguration">note options</a> to be changed.
                                                <br><a href="#doc_changeoptions">View an example</a>
                                            </td>
                                        </tr>

                                            <td>$(id).postitall('hide');</td>
                                            <td>Hide note.
                                                <br><span class="label label-info">Global call</span> If you don't specify the note <b>id</b>, all the notes of the page will be hidden.
                                                <br><a href="#doc_showhide1">View an example</a>
                                            </td>
                                        </tr>
                                        <tr>
                                            <th scope="row">Show note</th>
                                            <td>
                                                $.PostItAll.show();<br>
                                                $.PostItAll.show(id);
                                            </td>
                                            <td>$(id).postitall('show');</td>
                                            <td>Show note.
                                                <br><span class="label label-info">Global call</span> If you don't specify the note <b>id</b>, all the notes of the page will be shown.
                                                <br><a href="#doc_showhide1">View an example</a>
                                            </td>
                                        </tr>
                                        <tr>
                                            <th scope="row">Destroy note</th>
                                            <td>$.PostItAll.destroy(inline, storage, domain);</td>
                                            <td>$(id).postitall('destroy');</td>
                                            <td>Destroy a specific note or all the notes of your page.
                                                <br><span class="label label-info">Global call</span> If you don't specify the note <b>id</b>, all the notes of the page will be destroyed.
                                                <br>If <b>inline</b> is set to true, only visible notes will be removed (by default true).
                                                <br>If <b>storage</b> is set to true all stored notes will be deleted (by default true).
                                                <br>If <b>domain</b>
 is set to a valid domain name, only the notes created in this domain 
will be deleted. By default storage is local, so this options is useless
 in this conditions because you only see the notes in the current 
domain.
                                                <br><span class="label label-info">Plugin call</span> Only the note with the specified <b>id</b> will be removed.
                                                <br><a href="#doc_showhide1">View an example</a>
                                            </td>
                                        </tr>
